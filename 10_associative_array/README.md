10_associative_array
=====================

```php
$person = [
  'tom' => [
    'name' => 'Tom Snow',
    'age' => 32,
  ],
  'michael' => [
    'name' => 'Michael Wall',
    'age' => 45,
  ],
  'annie' => [
    'name' => 'Annie Liearea',
    'age' => 23,
  ],
];

$person['tom']['name'];

$person['michael'];

$person['annie']['age'];

```
